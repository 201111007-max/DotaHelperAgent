前端样式优化 - 当前界面分析

DotaHelperAgent 当前前端界面存在的问题和改进方向

当前界面问题

主要问题

  • 配色过于通用：渐变紫蓝色（#667eea → #764ba2）和粉红色（#e94560）缺乏 Dota 2 特色
  • 布局单调：单栏聊天 + 侧边栏，缺乏层次感
  • 消息展示简陋：纯文本气泡，无 Markdown 渲染，工具调用过程无结构化展示
  • 头部区域浪费空间：固定标题栏占据空间但信息量低
  • 侧边栏交互生硬：按钮切换式侧边栏，缺乏导航感
  • 缺少品牌元素：无 Dota 2 标志性视觉元素
  • 响应式支持差：min-width: 400px 硬编码，移动端不可用

Dota 2 官方设计语言

Dota 2 官方网站设计特征

品牌色

#BF2E1A
#1a1a1a

设计特征

  • 深黑背景 + 暗红强调色
  • 金色/铜色装饰线和分割
  • 锐利的边角（非圆角）
  • 厚重的字体和阴影
  • 石质/金属质感纹理
  • 英雄肖像作为视觉焦点

排版风格

  • 大标题 + 粗体无衬线字体
  • 全大写英文标题
  • 宽行距，强对比
  • 暗色背景 + 亮色文字
  • 红色/金色高亮关键词

优化方向

A

全面 Dota 2 官方风格

完全复刻 Dota 2 官方网站设计语言：深黑背景、暗红强调、金色装饰、锐利边角、石质纹理。视觉冲击力强，品牌一致性最高。

B

Dota 2 风格 + 现代融合

保留 Dota 2 的核心视觉元素（暗红+金色+深黑），但融入现代 UI 设计：适度圆角、更好的间距、流畅动画。兼顾品牌感和易用性。

C

轻量 Dota 2 主题

以现代暗色 UI 为基础，仅通过配色和少量装饰元素体现 Dota 2 风格。开发成本最低，但品牌辨识度较弱。
